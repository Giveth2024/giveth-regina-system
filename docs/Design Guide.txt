# Regina POS High-Velocity System

## Frontend Design Guide

==================================================

1. DESIGN PHILOSOPHY
   ==================================================

Regina POS should feel:

* Fast
* Modern
* Professional
* Clean
* Easy to understand
* Comfortable for long hours of use

Avoid unnecessary animations, excessive colors, or cluttered layouts.

Every screen should help the cashier or administrator complete a task quickly.

==================================================
7. INPUT DESIGN
===============

Dark background

Thin border

Rounded corners

Good padding

Example

Background

#111111

Border

#2A2A2A

---

==================================================
8. CARDS
========

Every major section should use cards.

Cards contain

Title

Description

Content

Optional action buttons

Cards should have

Dark background

Rounded corners

Thin border

Consistent padding

---

==================================================
9. TABLES
=========

Tables are the most important component.

Requirements

Sticky header

Alternating row colors

Hover effect

Compact rows

Search above table

Filters above table

Pagination below table

Columns should never feel cramped.

---

==================================================
10. ICONS
=========

Use one icon style consistently.

Recommended

Heroicons

Lucide

Font Awesome

Icons should be simple.

Avoid colorful icons.

---

==================================================
11. PAGE LAYOUT
===============

Every page should follow

Header

↓

Toolbar

↓

Search

↓

Filters

↓

Main Content

↓

Pagination

Do not mix unrelated components together.

13. FILTERS

Use dropdowns whenever possible.

Item Type

==================================================
15. STATUS COLORS
=================

Success

Green

Warning

Yellow

Danger

Red

---

==================================================
16. MODALS
==========

Dark background

Rounded corners

Clear title

Large close button

Primary action button

Secondary cancel button

---

==================================================
17. ANIMATIONS
==============

Keep animations subtle.

Allowed

Fade

Scale

Slide

Duration

150ms–250ms

Avoid flashy animations.

Speed is more important.

---

==================================================
18. RESPONSIVENESS
==================

Desktop first.

Support

Desktop

Laptop

Tablet

Mobile

Tables may scroll horizontally on smaller screens.

---

==================================================
19. LOADING STATES
==================

Always show loading feedback.

Use the Regina POS loader consistently.

Avoid blank pages.

---

==================================================
20. EMPTY STATES
================

Instead of blank tables

Show

Icon

Friendly message

Optional action button

Example

"No products found."

---

==================================================
21. DESIGN PRINCIPLES
=====================

Consistency over creativity.

Every page should feel like part of the same application.

Users should immediately know

where they are,

what they can do,

and what to do next.

The interface should prioritize speed, readability, and efficiency for cashiers, managers, and administrators who use the system throughout the day.
